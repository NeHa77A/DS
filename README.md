### Setup
Set the mpj 
```
set MPJ_HOME=C:\Users\Admin\OneDrive\Desktop\DS\mpj-v0_44
```

Set the path to bin
```
set PATH=%MPJ_HOME%\bin;%PATH%
```

compile the java
```
javac -cp "%MPJ_HOME%\lib\mpj.jar" ArrSum.java
```
run 
```
"%MPJ_HOME%\lib\mpjrun.bat" -np 4 ArrSum 
```
